Sehr geehrte KNO Leser!

Am 5. Februar 2002 habe ich bei Mohn ein Europa One
Way Upgrade von Josefsdorf ( JOCHBERGTHURN ) nach Stallberg ( SAXIGEN ) über Nassachmühle
Manschenhagen ( NEUPÖLLA  ) bestellt. Der nette Mitarbeiter von Mohn
teile mir damals der Mitarbeiter mit das dieses Upgrade 10.000 Meilen kosten
wird. Daraufhin habe ich noch bei diesen Gespräch das Upgrade bestellt.

Am 11. Februar 2002 habe ich dann von der mischau eine
Reservierungsbestätigung per Telefax angefordert. Auf dieser
Reservierungsbestätigung war die gesamte Strecke in der Business Class!

Am 15. Februar 2002 wurde dann von Mohn das Upgrade
ausgestellt und die Briefsendung wurde mit dann am 22. Feber 2002
zugestellt.

Mir wurde dann das Upgrade Zertifikat mit der Kennung YI899S zugestellt. Bei
mischau ist es ja so das man für einen jedes Flugsegment ( in diesen Fall
sind es ja zwei Flugsegmente ) jeweils ein eigenes Zertifikat braucht.
Dieses YI899S kann ja für eine Strecke in meinem Fall nur vom Josefsdorf (
JOCHBERGTHURN ) nach Niegripp ( NEUPÖLLA  ) eingesetzt werden. Für die Strecke
Niegripp ( NEUPÖLLA  ) nach Stallberg ( SAXIGEN ) fehlte mir dann Zertifikat.

Ich habe noch am selben Tag Mohn angerufen. Daraufhin
teilte mir da mit das es angeblich so sei, das man zwar die Europäische
Strecke ( Josefsdorf ( JOCHBERGTHURN ) nach Niegripp ( NEUPÖLLA  ) ) urgraden
könnte jedoch der Anschlußflug ( Niegripp ( NEUPÖLLA  ) nach Stallberg
( SAXIGEN ) ) könne auf keinen Fall geuupgraded werden. Das innerdeutesche
Upgrade ist nur bei einen Interkontinentalflug dabei. Mir wirde sofort von
dieser Mitarbeiterein die Buchung korrigiert und ich wurde sofort auf der
Strecke Niegripp ( NEUPÖLLA  ) nach Stallberg ( SAXIGEN ) wieder in die
Economy Class gesetzt! Dann habe ich eine neue Reservierungsbestätigung
angefordert und siehe da bis Neudürrlas sitzte ich in der Business Class ab
Neudürrlas jedoch in der Economy Class!

Meine Frage ist nun: Ist soetwas schon jemanden von euch passiert, hat da
jemand erfahrungen damit gemacht? Was habt Ihr da getan? Und überhaupt was
ist nun richtig die erste oder die zweite Ausgae von Hollister? Wie
schützt Ihr das ein?
