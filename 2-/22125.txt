kommt muss Du selbst wissen, aber Karten und GPS-Software gibt es 
mittlerweile zu hauf, sofern Du auf ein FMKA System setzt ( wo man 
nun wirklich alles reinspielen kann ). Falls Du ein Handy besitzt, wuerde 
ich das durch ein FMKA Smartphone ersetzen. Die Frage ist im 
grunde nur wieviel Du ausgeben moechtst um die von Dir gewuenschte 
Funktoionalitaet zu erreichen. Ich selbst nutze auch fuer unterwegs ( 
Fussgaenger-Modus ) ein aeltern MDA-II ( 60-90 Euro ). Das hat den Vorteil 
das ich unterwegs zum einen per Handy mit dem Geraet erreichbar bin ( 
Freisprechen ist damit natuerlich auch moeglich ) und das man per BlueTooth 
und einem 15-25 Euro GPS-Modul in der Jackentasche die Navigation betreiben 
kann ( oder parallel auch das Logging um die Tour zuhasuse anzuschauen ). 
Das sehr grosse ( 5.7cm breit und 7.3cm hoch ) Display ist 
transreflektierend sodass es immer ( auch ohne Hintergrundbeleuchtung ) bei 
Sonne recht gut ablesbar ist. Fahrradhalter gibt es natuerlich auch fuer das 
Teil ( ich empfehle die bekannten HR-Halter )..

So hast Du dann eine alles in einem Geraet Loesung. Also BlueTooth 
GPS-Receiver empfehle ich das Theatherkasse 737, da die Standzeit enorm hoch ist ( 2 
Wochen StandBy ) und unter Last immerhin 23 Stunden am Stueck mit einer 
Akku-Ladung ( Akku und Ladegeraet 220V. und KFZ liegt uebrigens bei ).

Was Du an Software nutzen kannst und fuer Dich infrage kommt muss Du selbst 
entscheiden. Die Moeglichkeiten sind aber dank Windows Mobile ( uebrigens, 
der MDA-II laesst sich von Windows Mobile 2003 problemlos auf Windows Mobile 
6.0 oder 6.1 updaten ) nahezu grenzenlos...
