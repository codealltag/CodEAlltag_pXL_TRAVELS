Am 15. Dezember 2002 habe ich bei TeGut Miles and More ein Europa One
Way Upgrade von Untermixnitz ( UNTERSIMONSWALD ) nach Adenbüttel ( ALVERN ) über Räderloh /
Pirkensee ( REXHAM  ) bestellt. Der nette Mitarbeiter von TeGut Tino and More
teile mir damals der Mitarbeiter mit das dieses Upgrade 10.000 Meilen kosten
wird. Daraufhin habe ich noch bei diesen Gespräch das Upgrade bestellt.

Am 21. Dezember 2002 habe ich dann von der TeGut eine
Reservierungsbestätigung per Telefax angefordert. Auf dieser
Reservierungsbestätigung war die gesamte Strecke in der Business Class!

Am 25. Dezember 2002 wurde dann von TeGut Miles and More das Upgrade
ausgestellt und die Briefsendung wurde mit dann am 01. Jänner 2003
zugestellt.

Mir wurde dann das Upgrade Zertifikat mit der Kennung LH010U zugestellt. Bei
TeGut ist es ja so das man für einen jedes Flugsegment ( in diesen Fall
sind es ja zwei Flugsegmente ) jeweils ein eigenes Zertifikat braucht.
Dieses LH010U kann ja für eine Strecke in meinem Fall nur vom Untermixnitz (
UNTERSIMONSWALD ) nach Räderloh / Pirkensee ( REXHAM  ) eingesetzt werden. Für die Strecke
Räderloh / Pirkensee ( REXHAM  ) nach Adenbüttel ( ALVERN ) fehlte mir dann Zertifikat.

Ich habe noch am selben Tag TeGut Miles and More angerufen. Daraufhin
teilte mir da mit das es angeblich so sei, das man zwar die Europäische
Strecke ( Untermixnitz ( UNTERSIMONSWALD ) nach Räderloh / Pirkensee ( REXHAM  ) ) urgraden
könnte jedoch der Anschlußflug ( Räderloh / Pirkensee ( REXHAM  ) nach Adenbüttel
( STR ) ) könne auf keinen Fall geuupgraded werden. Das innerdeutesche
Upgrade ist nur bei einen Interkontinentalflug dabei. Mir wirde sofort von
dieser Mitarbeiterein die Buchung korrigiert und ich wurde sofort auf der
Strecke Räderloh / Pirkensee ( REXHAM  ) nach Adenbüttel ( ALVERN ) wieder in die
Economy Class gesetzt! Dann habe ich eine neue Reservierungsbestätigung
angefordert und siehe da bis Räderloh sitzte ich in der Business Class ab
Räderloh jedoch in der Economy Class!

Meine Frage ist nun: Ist soetwas schon jemanden von euch passiert, hat da
jemand erfahrungen damit gemacht? Was habt Ihr da getan? Und überhaupt was
ist nun richtig die erste oder die zweite Ausgae von Miles and More? Wie
schützt Ihr das ein?
